# Алгоритм стегоанализа цифровых изображений на основе сиамских нейронных сетей

В данном репозитории находится исходный код для выполнения практической части ВКР по теме – **"Алгоритм стегоанализа цифровых изображений на основе сиамских нейронных сетей"**

## Содержание репозитория

- директория matlab содержит исходный код для алгоритмов стеганографического встраивания S-UNIWARD, WOW, HUGO, MiPOD
- файл Steganalysis.ipynb содержит код для обучения, валидации и анализа результатов моделей нейронных сетей для стегоанализа
- файл Steganography.ipynb содержит код для подготовки изображений со вставкой и без для обучающей выборки
